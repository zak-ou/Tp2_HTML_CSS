1.2 Spécificité

Sélecteur Spécificité (id, class, tag) Valeur
p(0, 0, 1)1
.text(0, 1, 0)10
p.text(0, 1, 1)11
#main(1, 0, 0)100
#main .text p(1, 1, 1)111
Si deux règles ont la même spécificité : la dernière déclarée dans le CSS gagne (ordre de cascade).
Pourquoi éviter !important ? Il casse la cascade naturelle du CSS, rend le code difficile à maintenir et à déboguer.
